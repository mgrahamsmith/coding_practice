1. Compiling and running a C++ Program
--------------------------------------------------------------------------------
Install g++ compiler in (assuming Ubuntu environment)::
    
    $ sudo apt install g++

Compile the program::

    g++ -o hello_world.out hello_world.cpp

Running the program::

    $./hello_world.out
